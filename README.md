# Battleships

JavaScript, HTML and CSS Battleship game.

The game was successfully deployed to a server, which was the main goal of this project.

## Modifications

- All ships must be placed before start.

- Not stacking and overlapping ships logic.

- Reset board button.

- Disabling shooting at the same place twice.

- Adding bonus shot after a successful hit.

- Easy and Medium single player difficulty.
